Talkprogram
==================
コンピュータと会話形式で晩御飯のメニューを決めるプログラム（？）です。


### 操作の仕方 ###
------
トークのはじめは、「ただいま」と入力してから開始してください。  
基本的な流れは、コンピュータが話しかけてきたセリフに対して、自分なりのセリフを入力し、発言することで会話を進めていきます。  
「はい」や「いいえ」などの言葉には反応しないので注意してください。  
コンピュータから提案されたメニューの画像をクリックすると、別ウィンドウでレシピが紹介されているページに飛びます。
